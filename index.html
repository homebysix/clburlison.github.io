10.12.4 locationd

Apr 03, 2017

New with 10.12.4, Apple has decided to protect the locationd LaunchDaemon with System Integrity Protection (SIP). If you are not aware the locationd daemon is used for Location Services. This is the process that drives: FindMyMac, “Set time zone automatically using current location”, it allows applications to locate your computer, and more. Some points to take away from this change: this modification happened during a point release this path is not in /System/Library/Sandbox/rootless.
